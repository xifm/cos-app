🚀 CosApp 纯净版 (个人逆向折腾记录)
> “因为受够了原版 App 满天飞的广告和 VIP 限制，作为一个强迫症，我决定自己动手捏一个。”
> 
这是一个基于纯前端技术手搓的第三方客户端。没有后端，没有数据库，所有的接口解析和数据解密全靠浏览器里的 JS 硬干。写这个项目纯粹是为了练习网络抓包和逆向工程，顺便给自己弄个干干净净的看片环境。
💡 我做了什么？
 * 🚫 彻底干掉广告：纯净的 HTML + CSS 瀑布流，没有任何恶心的弹窗、遮罩和启动页。
 * 🔓 破解底层加密：原版 API 返回的全是 AES 加密的 JSON 乱码。我把它的内部密钥（多重 MD5）给扒了出来，实现了纯前端实时秒解密。
 * 🕷️ 直捣黄龙的解析：跳过表层列表的伪装，直接写了个后台特工并发请求详情页，把带着防盗链 Token 的真实 CDN 下载链接 (.m3u8 / .mp4) 剥得干干净净。

🛠️ 食用指南 (怎么跑起来？)
这个项目极其轻量，本质上就是一个 index.html，你想怎么折腾都行：
1. 最懒的玩法（本地直开）
直接把代码克隆或下载到本地，双击 index.html 用浏览器打开就能跑。
(注：由于浏览器的同源策略，如果你在电脑上直开，记得给浏览器装个 Allow CORS 之类的跨域插件。)
2. 白嫖云端（推荐极客玩法）
直接 Fork 本仓库，开启 GitHub Pages，白嫖一个永久在线的 https 链接。完美绕过本地跨域限制，而且只要在云端改改代码，你的手机端就能无缝热更新。
(建议再去买个便宜的顶级域名绑上去，逼格瞬间拉满。)
3. 套壳成专属 App（我的日常用法）
把上面部署好的 HTTPS 网页链接，丢进 Fusion App (手机端) 或者 HBuilderX (电脑端) 里。关掉系统顶栏，开启沉浸式状态栏，换上自己找的 Logo，一键打包成 APK。装到手机上，这就是你独一无二的私人无广告 App。
