🚀 CosApp 终极聚合版 (CosApp Ultimate)
这是一个基于纯前端技术栈打造的第三方、无广告、纯净版 CosApp 影视聚合客户端。本项目通过逆向解析官方 API 接口，实现了数据的全自动解密与真实播放链接的提取，并提供优雅的瀑布流 UI 体验。
✨ 核心特性 (Features)
 * 🔓 全自动数据解密：内置智能解密引擎，自动适配基础版、加强版、双重MD5等多种 AES 锁型，瞬间还原被加密的 JSON 响应数据。
 * 🎬 全频道聚合解析：完美对接官方接口，支持“最新”、“热门”、“独家”、“FC2”、“AV”、“同人”等多频道无缝切换。
 * ⚡ 真实地址嗅探：突破列表页信息壁垒，后台并发执行详情页穿透，智能提取并剥离带有防盗链 Token 的 CDN 多线路 m3u8 / mp4 真实播放地址。
 * 🔒 最高权限安全锁：内置页码拦截机制，严格锁定仅允许访问第 1 和第 2 页，防止过度请求触发服务器风控。
 * 📱 极致 UI 体验：采用现代化的 CSS Grid 瀑布流布局，自适应电脑与手机屏幕；搭配深色沉浸式主题，观感超越原生 App。
🛠️ 技术栈 (Tech Stack)
 * 前端架构：HTML5 + CSS3 + Vanilla JavaScript (原生 JS)
 * 加密核心：crypto-js.min.js (MD5 密钥生成 + AES 数据解密)
 * 网络请求：Fetch API (原生异步并发请求)
 * 打包部署：支持 GitHub Pages 云端部署 / Fusion App 本地套壳封装
📖 部署与使用指南 (How to Use)
方案一：本地浏览器直开 (极速体验)
 * 下载本仓库的 index.html 文件。
 * 确保你的浏览器允许跨域请求（CORS），或者安装相应的浏览器插件。
 * 双击运行 index.html 即可开始观影。
方案二：云端部署 (推荐，支持热更新)
 * 将本项目 Fork 或上传至你自己的 GitHub 仓库。
 * 在仓库的 Settings -> Pages 中，将分支设置为 main。
 * 等待 1 分钟，即可通过 GitHub 提供的 https://用户名.github.io/仓库名/ 链接随时随地在线访问。
 * (可选) 支持绑定自定义域名，打造专属私人影视站。
方案三：打包 Android App (终极形态)
 * 在手机上安装 Fusion App 或在电脑上使用 HBuilderX。
 * 创建“空白模板”项目。
 * 将入口链接设置为你部署好的云端 https:// 地址。
 * 关闭系统应用栏、侧滑栏，开启沉浸式状态栏。
 * 替换专属 Logo 图标，一键打包生成 APK。
